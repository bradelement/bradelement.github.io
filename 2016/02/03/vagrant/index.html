vagrant/docker安装小结

发表于 2016-02-03   |   分类于 tools   |  

vagrant:

vagrant init ubuntu/trusty64
vagrant up

vagrant ssh --貌似等价于ssh 127.0.0.1 -p 2222 用户名/密码都是vagrant
vagrant package --不打包貌似会丢安装好的东西

docker:

wget -qO- https://get.docker.com/gpg | sudo apt-key add -
wget -qO- https://get.docker.com/ | sh
