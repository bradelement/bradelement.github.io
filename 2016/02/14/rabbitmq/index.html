rabbitmq笔记

发表于 2016-02-14   |   分类于 tools   |  

线上要安装个rabbitmq，本机简单搭建并记录一下。

brew install rabbitmq
/usr/local/sbin/rabbitmq-server #可以加到path里

有教程，可参考具体代码
https://www.rabbitmq.com/getstarted.html

rabbitmqctl add_user username password
rabbitmqctl set_user_tags username none/management/policymaker/monitoring/administrator
rabbitmqctl list_users

rabbitmqctl add_vhost hostname
rabbitmqctl list_vhosts

rabbitmqctl set_permissions  -p /hostname username '.*' '.*' '.*'
rabbitmqctl list_permissions -p /hostname
rabbitmqctl list_user_permissions username
谢赏~
