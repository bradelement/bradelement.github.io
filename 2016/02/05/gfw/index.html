翻越伟大的gfw

發表於 2016-02-05   |   分類於 tools   |  

之前一直使用lantern，浏览器上网无压力，wget/git等仍然被和谐。
最近被介绍了shadowsocks和proxychains两大神器，简单记录下。

shadowsocks目前先安装了gui版，命令行版暂时研究失败(ss-local/ss-server)… 后续补上。
gui版直接输入服务器端用户名，密码，加密方式等即可。

proxychains主要用来做console里的socks5代理。

brew install proxychains-ng

理论上直接brew安装即可，修改配置文件，增加socks5 ip port配置即可。

proxychains4 curl www.youtube.com

可惜在我的小mac上不能成功。
查阅资料后，目前结论是mac自带的curl, git版本太低，不支持proxychains，brew install curl/git即可。
安装后curl不能覆盖原curl(求教各位大神如何覆盖系统自带的curl…) git亲测可行… 以后有新发现再行补充~

proxychains4 wget www.youtube.com
proxychains4 git clone blabla
